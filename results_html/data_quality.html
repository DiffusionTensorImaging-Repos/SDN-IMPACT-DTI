Correlated with each other?

No. Pearson r = -0.042, p = 0.759 (n=56). The two memory scores are essentially independent — a subject good at social memory isn't necessarily good at monetary memory. This supports treating the social and monetary tract findings as a genuine dissociation, not two views of one signal.

2 · How d′ was calculated

Standard signal-detection theory. "Old" items = faces you chose at encoding (received feedback); "new" items = faces you saw but did not choose (foils).
hit rate (H) = # "remember" responses to OLD faces / # OLD faces false-alarm (F) = # "remember" responses to NEW faces / # NEW faces d′ = z(H) − z(F) # z = inverse-normal (probit) bias c = −½ · (z(H) + z(F)) # criterion / response bias
Our independent recomputation of d′ from the raw trial files reproduces the analysis values almost exactly:
