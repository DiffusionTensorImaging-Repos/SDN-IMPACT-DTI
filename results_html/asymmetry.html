Is the hemispheric “flip” a laterality effect? Tract balance vs. tract level

Social memory loaded on the left tract, monetary on the right — opposite signs. Is that a left/right balance phenomenon, or something else?  ·  ← Results Explorer

The question

The two headline accuracy findings point in opposite directions: social d′ tracks the left VTA→HPC tract (denser → better), monetary d′ tracks the right tract (denser → worse). Read side-by-side, that looks like a laterality story:

Hypothesis: do mothers whose left tract is stronger than their right tend to be better at social memory (relative to monetary), while mothers with a stronger right tract are better at monetary? If so, the flip is one real thing — the left/right balance of this pathway sets which kind of memory you are tuned for.

This is worth testing directly, because two opposite-signed correlations sitting next to each other is not, by itself, evidence of a coherent asymmetry. It could be a genuine balance effect — or an artifact of analysing four cells (L/R × social/monetary) separately.

How we tested it

Tract balance (predictor). For each mother, a laterality index on the deep-white-matter mid-tract (nodes 25–75, avoiding endpoint partial-volume) — computed on both NDI (density) and FA (coherence). Using the whole mid-tract, not the specific significant cluster nodes, so we are not re-selecting the nodes that produced the original finding.

Memory balance (outcome). Social and monetary d′ are first z-scored within-sample (their raw scales differ), then differenced: z(social) − z(monetary) = a mother's relative social-vs-monetary memory advantage. This is a meaningful dimension precisely because the two d′s are essentially uncorrelated (r ≈ −0.04) — the difference is signal, not shared variance.

Laterality index LI = (Left − Right) / (Left + Right) (mid-tract, NDI & FA) Memory balance = z(social d′) − z(monetary d′) Test: memory_balance ~ LI + ICV + motion + age + streamline-count-LI + length-LI

Two task non-completers excluded (social d′ enters the outcome). Predictor z-scored (β = standardized). Streamline count/length asymmetry included as nuisance covariates — controlling the lateralized confound, not just the global one.

Tract balance does not predict memory balance. Neither density asymmetry (β=+0.104, p=0.6952) nor coherence asymmetry (β=+0.225, p=0.4308) is associated with whether a mother is relatively better at social vs. monetary memory. Left-dominant mothers are not tuned toward social memory. The flip is not a left/right competition.

Result 2 — it is the level, not the balance SUPPORTED

If it is not the difference between hemispheres, what predicts the social-vs-monetary tradeoff? The total — bilateral tract level (Left + Right):

Higher overall bilateral VTA→HPC density and coherence shift the whole system toward social and away from monetary memory (NDI β=+0.550, p=0.0291; FA β=+0.650, p=0.0133). This is why the two original findings had opposite signs: it was never a contest between hemispheres — more tract (on either side) pushes memory toward the social domain and away from the monetary one, so the difference (social − monetary) rises with the sum (L + R), not with the imbalance (L − R).

What this changes about the story

Before: “social memory is left-lateralized, monetary is right-lateralized” — a puzzling, fragile double dissociation that reads like it might be noise.

After: “the overall strength of the VTA→HPC pathway biases memory toward the social domain at the expense of the monetary one.” One bilateral individual-differences dimension, not a hemispheric flip. That is a cleaner and more defensible claim — and it reframes the odd negative monetary effect not as a hemisphere-specific quirk but as the other end of a single tradeoff axis.

Honest caveats

This is a reframing of the same data, not independent replication. The bilateral-level effect is the single interaction that the two original cluster findings logically imply; testing it here characterizes the phenomenon correctly but does not add out-of-sample confirmation. A regression on the difference score is the corresponding domain × predictor interaction test — the balance-null and level-effect are two readouts of the same model, not independent experiments.
Small sample, mid-range p-values. n≈40 after covariate/d′ listwise deletion; the level effects are p≈0.01–0.03, real but not bulletproof. The direct-question answer (balance = null) is robust across both metrics; the “it’s the sum” conclusion follows algebraically from social(+) and monetary(−) sharing a density substrate and is corroborated empirically here.
