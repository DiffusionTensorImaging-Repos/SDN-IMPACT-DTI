Superseded — the “flip” was a sample artifact Archived analysis

This page tested whether an apparent social-left / monetary-right sign-flip was a laterality effect.  ·  ← Results Explorer

⚠ This analysis has been superseded

It was built on the earlier result that monetary d′ tracked the right VTA→HPC tract (denser → worse), forming an apparent left/right sign-flip against the social finding. On the corrected roster (55 mothers: 2 pilot scans removed, maternal ages recovered), no monetary tract cluster survives FWE correction — the right-negative monetary effect was a sample artifact. There is no flip to explain. (This particular difference-score model runs at n=52, after also dropping 2 task non-completers and 1 mother without a d′ score.)

What still holds: higher bilateral VTA→HPC density is associated with relatively better social than monetary memory — but because the monetary side is now null, that difference is driven entirely by the social finding, so this page adds nothing independent beyond the main social results. It is kept only as a record of the laterality check. Numbers below are recomputed on the corrected sample.

The question

(Framed in the past tense — this was the reasoning at the time, before the correction.) In the earlier run the two headline accuracy findings appeared to point in opposite directions: social d′ tracked the left VTA→HPC tract (denser → better), and monetary d′ appeared to track the right tract (denser → worse). Read side-by-side, that looked like a laterality story:

Hypothesis (as posed then): did mothers whose left tract was stronger than their right tend to be better at social memory (relative to monetary), while mothers with a stronger right tract were better at monetary? If so, the flip would be one real thing — the left/right balance of this pathway setting which kind of memory you are tuned for.

We tested it directly. As the banner above notes, the premise itself did not survive correction (the monetary-right effect was an artifact), but the laterality test is recorded below for completeness.

How we tested it

Tract balance (predictor). For each mother, a laterality index on the deep-white-matter mid-tract (nodes 25–75, avoiding endpoint partial-volume) — computed on both NDI (density) and FA (coherence). Using the whole mid-tract, not the specific significant cluster nodes, so we are not re-selecting the nodes that produced the original finding.

Memory balance (outcome). Social and monetary d′ are first z-scored within-sample (their raw scales differ), then differenced: z(social) − z(monetary) = a mother's relative social-vs-monetary memory advantage. This is a meaningful dimension precisely because the two d′s are essentially uncorrelated (r ≈ −0.09, p = 0.53) — the difference is signal, not shared variance.

Laterality index LI = (Left − Right) / (Left + Right) (mid-tract, NDI & FA) Memory balance = z(social d′) − z(monetary d′) Test: memory_balance ~ LI + ICV + motion + age + streamline-count-LI + length-LI

Two task non-completers excluded (social d′ enters the outcome). Predictor z-scored (β = standardized). Streamline count/length asymmetry included as nuisance covariates — controlling the lateralized confound, not just the global one.

Left-dominant mothers are not tuned toward social memory. The flip is not a left/right competition.

Higher overall bilateral VTA→HPC density and coherence shift the whole system toward social and away from monetary memory (NDI β=+0.522, p=0.016; FA β=+0.514, p=0.0209). This is why the two original findings had opposite signs: it was never a contest between hemispheres — more tract (on either side) pushes memory toward the social domain and away from the monetary one, so the difference (social − monetary) rises with the sum (L + R), not with the imbalance (L − R).

What this changes about the story

Before: “social memory is left-lateralized, monetary is right-lateralized” — a puzzling, fragile double dissociation that reads like it might be noise.

After: “the overall strength of the VTA→HPC pathway biases memory toward the social domain at the expense of the monetary one.” One bilateral individual-differences dimension, not a hemispheric flip. That is a cleaner and more defensible claim — and it reframes the odd negative monetary effect not as a hemisphere-specific quirk but as the other end of a single tradeoff axis.

Honest caveats

This is a reframing of the same data, not independent replication. The bilateral-level effect is the single interaction that the two original cluster findings logically imply; testing it here characterizes the phenomenon correctly but does not add out-of-sample confirmation. A regression on the difference score is the corresponding domain × predictor interaction test — the balance-null and level-effect are two readouts of the same model, not independent experiments.
Modest sample, mid-range p-values. n=52 (corrected roster: 55 mothers, 2 pilots removed, 12 recovered ages, minus 2 task non-completers and 1 without d′); the level effects are p≈0.01–0.03, real but not bulletproof. The direct-question answer (balance = null) is robust across both metrics; the “it’s the sum” conclusion follows algebraically from social(+) and monetary(−) sharing a density substrate and is corroborated empirically here.
